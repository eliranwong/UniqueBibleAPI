# INDEX

Keyword: INDEX

Keyword description:

* Retrieve available resource information for studying a bible verse.

Parameter: BIBLE_REFERENCE [required]

Parameter description:

* Specify a verse reference.

# Command Format

INDEX:::[BIBLE_REFERENCE]

# Examples

Command: `INDEX:::John 3:16`

URL: https://bible.gospelchurch.uk/json?cmd=INDEX:::John%203:16
