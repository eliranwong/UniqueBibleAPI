# COMPARECHAPTER

Keyword: COMPARECHAPTER

Keyword description:

* Compare a bible chapter in different bible versions.

Parameter: BIBLE_VERSIONS [optional]

Parameter description:

* Specify bible versions for comparison.
* Use underscore `_` as a separator to specify multiple bible versions.
* Use favourite bible versions configured in `config.py` if this parameter is not given.

Parameter: BIBLE_CHAPTER [required]

Parameter description:

* Specify a bible chapter.

# Command Format

COMPARECHAPTER:::[BIBLE_VERSION]:::[BIBLE_CHAPTER]

# Examples

Command: `COMPARECHAPTER:::John 3`

URL: https://bible.gospelchurch.uk/json?cmd=COMPARECHAPTER:::John%203
