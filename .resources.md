# .resources

Keyword: .resources

Description:

* Retrieve information about available resources loaded in the web server.
* This command keyword takes no parameter.
* This command output in JSON format only with all UniqueBible App web API endpoints: `/html`, `/json`, `/plain`.
* This command applies to UniqueBible App running mode `http-server` only.

# Command Format

.resources

# Example

Command: `.resources`

URL: https://bible.gospelchurch.uk/json?cmd=.resources
