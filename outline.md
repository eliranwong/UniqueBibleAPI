# OUTLINE

Keyword: OUTLINE

Keyword description:

* Display all subheadings in a bible book.

Parameter: BIBLE_BOOK_NAME [required]

Parameter description:

* Specify a bible book name.

# Command Format

OUTLINE:::[BIBLE_BOOK_NAME]

# Examples

Command: `OUTLINE:::John`

URL: https://bible.gospelchurch.uk/json?cmd=OUTLINE:::John
