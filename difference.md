# DIFFERENCE

Keyword: DIFFERENCE

Keyword description:

* Show differences of verses between different bible versions.

Parameter: BIBLE_VERSIONS [optional]

Parameter description:

* Specify bible versions for comparison.
* Use underscore `_` as a separator to specify multiple bible versions.
* Use all bible versions configured in `config.py` if this parameter is not given.

Parameter: BIBLE_REFERENCES [required]

Parameter description:

* Specify a verse reference or multiple verse references.

# Command Format

DIFFERENCE:::[BIBLE_VERSIONS]:::[BIBLE_REFERENCES]

# Examples

Command: `DIFFERENCE:::KJV_ASV_WEB:::Joh 3:16`

URL: https://bible.gospelchurch.uk/html?cmd=DIFFERENCE:::KJV_ASV_WEB:::Joh%203:16
